How to Fix Common Issues

If API server is not running:
• Start your backend server on port 5188
• Check if the port 5188 is available
• Verify your backend configuration

If CORS errors occur:
• Add CORS headers to your API server
• Allow origin: http://localhost:8000
• Allow methods: POST, OPTIONS
• Allow headers: Content-Type

Example CORS configuration (Express.js): 
app.use(cors({
  origin: ['http://localhost:8000', 'http://127.0.0.1:8000'],
  methods: ['GET', 'POST', 'OPTIONS'],
  allowedHeaders: ['Content-Type', 'Authorization']
}));
