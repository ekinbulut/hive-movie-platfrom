How to Fix CORS in Your Backend

For ASP.NET Core, add to Program.cs: 
// Add CORS services
builder.Services.AddCors(options =>
{
    options.AddPolicy("AllowFrontend", policy =>
    {
        policy.WithOrigins("http://localhost:8000")
              .AllowAnyMethod()
              .AllowAnyHeader();
    });
});

// Use CORS middleware (before app.UseAuthentication())
app.UseCors("AllowFrontend");
